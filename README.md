# Portfolio Website

Simple personal portfolio website built with Next.js and Tailwind CSS.

It includes a landing section, profile, experience timeline, projects, skills, education, and contact CTA.

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- ESLint

## Platform / Services

- Deployment: Vercel
- Backend/Data services: Supabase

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- Main page content is in `app/page.tsx`.
- Global styles are in `app/globals.css`.
